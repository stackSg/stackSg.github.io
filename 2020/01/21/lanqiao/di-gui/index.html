  1. 小白上楼梯（递归设计）

    • 小白正在上楼梯,楼梯有n阶台阶,小白一次可以上1阶, 2阶或者3阶,实现一个方法,计算小白有多少种走完楼梯的方式。

    • 实现：

      static int fun(int n){
  if (n == 0) return 1;
  if (n == 1) return 1;
  if (n == 2) return 2;

  return fun(n - 1) + fun(n - 2) + fun(n - 3);
}

    • 迭代实现：

      static int countWays2(int n){
  if (n <= 0)    return 0;
  if (n < 3)     return n;
  if (n == 3) return 4;

  int a = 1;
  int b = 2;
  int c = 4;
  int d = 0;
  for (int i = 4; i <= n; i++){
      d = a + b + c;
      a = b;
      b = c;
      c = d;
  }

  return d;
}

  2. 旋转数组的最小数字（改造二分法）

    • 把一个数组最开始的若干个元素搬到数组的末尾,我们称之为数组的旋转。输入一个递增排序的数组的一个旋转,输出旋转数组的最小元素。例如数组{3,4,5,1,2}为{1,2,3,4,5}的一个旋转,该数组的最小值为1.

    • 实现：

      //考察灵活运用二分法
static int min(int[] arr){
  int begin = 0;
  int end = arr.length - 1;

  //如果没有旋转则返回最小的
  if (arr[begin] < arr[end]){
      return arr[begin];
  }

  //相较于二分查找，此处不是找某个数字，而是查找到数组旋转的边界
  //因此剩两个数的时候可以确定边界
  while (begin + 1 < end){
      int mid = (begin + end) / 2;

      //判断有序的为哪一边，边界总是在无序的一边
      //{1, 0, 1, 1, 1}数组存在多个重复值的情况下，无法定位哪边有效，规则失效
      if (arr[mid] >= arr[begin]){
          begin = mid;
      }
      else{
          end = mid;
      }
  }

  //定位到旋转的边界后，递增数组的首个元素一定为后面的那一个
  return arr[end];
}

  3. 在有空字符串的有序字符串中查找

    • 有个排序后的字符串数组,其中散布着一些空字符串 ,编写一个方法，找出给定字符串(肯定不是空字符串)的索引。

    • 实现：

      static int search(String[] arr, String p){
  int begin = 0;
  int end = arr.length - 1;

  while (begin <= end){
      int mid = (begin + end) / 2;

      //如果碰到空串则mid右移
      while (arr[mid] == ""){
          mid++;
          //如果mid超出查找边界则查找失败
          if (mid < begin || mid > end){
              return -1;
          }
      }

      if (arr[mid].compareTo(p) > 0){
          end = mid - 1;
      }
      else if (arr[mid].compareTo(p) < 0){
          begin = mid + 1;
      }
      else{
          return mid;
      }
  }

  return -1;
}

  4. 最长连续递增子序列（部分有序）

    • (1,9,2,5,7,3,4,6,8,0)中最长的递增子序列为(3,4,6,8)。
    • 实现：
      public static void main(String[] args) {
  int[] arr = {1,9,2,5,7,3,4,6,8,9, 0};
  int i = 0;
  int j = i + 1;
  int max = 0;
  int maxI = 0;
  int maxJ = 0;
  while (i < arr.length && j < arr.length){
      while (arr[j - 1] < arr[j]){
          j++;
      }
      //维护最大长度
      if (max < j - i){
          max = j - i;
          maxI = i;
          maxJ = j - 1;
      }
      i = j;
      j = j + 1;
  }
  System.out.println(maxI + " " + maxJ);
}

  5. 设计一个高效的求n次幂的算法。

    • 实现：

      //以翻倍的方式求次幂
static int pow(int a, int n){
  if (n == 0){
      return 1;
  }
  int result = a;
  int c = 1;
  //如果翻倍之后小于指数，则继续翻倍
  while ((c << 1) < n){
      result *= result;
      c <<= 1;
  }

  //翻倍得到的结果*剩下的次方数继续翻倍
  return result * pow(a, n - c);
}
