  1. 1- 1000这1000个数放在含有1001个元素的数组中，只有唯一的一个元素值重复，其它均只出现一次。每个数组元素只能访问一次，设计一个算法，将它找出来;不用辅助存储空间，能否设计一个算法实现?

    • 解题思路：A^A=0

      int N = 1001;
int[] arr = new int[N];

//把1-（n - 1）初始化到数组中
for (int i = 0; i < N - 1; i++){
  arr[i] = i + 1;
}
//数组最后一个位置随机生成一个范围内的数字
arr[N - 1] = new Random().nextInt(N - 1) + 1;
//随机一个数组下标，将随机数和对应下标数字交换
int index = new Random().nextInt(N);
int temp = arr[N - 1];
arr[N - 1] = arr[index];
arr[index] = temp;

System.out.println(Arrays.toString(arr));
//解题思路 连续的异或运算能够消除重复 例如：A^B^B^A^C=C
//[1,2,3,...,K,...K,...,1000]^1^...^1000=K
int res = 0;
for (int i = 1; i <= N - 1; i++){
  res ^= i;
}

for (int i = 0; i < N; i++){
  res ^= arr[i];
}

System.out.println(res);

//方法二：使用数组下标做累加器，得到重复数字
int[] count = new int[N + 1];

for (int i = 0; i < N; i++){
  count[arr[i]]++;
}

for (int i = 0; i < N + 1; i++){
  if (count[i] == 2){
      System.out.println(i);
  }
}

  2. 请实现一个函数，输入一个整数，输出该数二进制表示中1的个数。例: 9的二进制表示为1001，有2位是1

     int num;

 Scanner s = new Scanner(System.in);
 num = s.nextInt();

 System.out.println(Integer.toBinaryString(num));

 // 解法1：整数每一位和1进行与运算判断这一位是不是零
 int count = 0;
 for (int i = 0; i < 32; i++) {
     if ((num & 1 << i) == (1 << i)) {
         count++;
     }
 }

 System.out.println(count);

 //解法2：整数最低位和1与运算，判断是否为1，整数右移继续
 count = 0;
 int temp = num;
 while (temp != 0){
     if ((temp & 1) == 1){
         count++;
     }
     temp >>= 1;
 }

 System.out.println(count);

 //解法3：(X-1)&(X)会将X最低位开始非0位以下的位置0
 count = 0;
 temp = num;
 while (temp != 0){
     temp = (temp - 1) & temp;
     count++;
 }

 System.out.println(count);

  4. 将整数的奇偶位互换。

     public static int swap(int n){
     //n&1010得到偶数位数字
     int a = (n & 0xaaaaaaaa);
     //n&0101得到奇数位数字
     int b = (n & 0x55555555);
     //偶数位右移1位^奇数位左移1位实现奇偶位交换
     return ((a >> 1) ^ (b << 1));
 }

  5. 给定一个介于0和1之间的实数，(如0.625) ，类型为double,打印它的二进制表示(0.101,因为小数点后的二进制分别表示0.5,0.25.0.125…..)。如果该数字无法精确地用32位以内的二进制表示，则打印“ERROR”。

     double n = 0.625d;

 StringBuilder sb = new StringBuilder("0.");

 while (n > 0){
     //乘二取整
     double t = n * 2;

     if (t >= 1){
         sb.append("1");
         n = t -= 1;
     }
     else{
         sb.append("0");
         n = t;
     }

     if (sb.length() == 34){
         System.out.println("ERROR");
         return;
     }
 }

 System.out.println(sb);

  6. 数组中只有一一个数出现了1次，其他的数都出现了k次，请输出只出现了1次的数。

    • 解题思路：k个k进制的数各位不进位相加结果为0

      int k;
Scanner s = new Scanner(System.in);
k = s.nextInt();

//随机得到题目要求的数组
int[] arr = new int[k * (k - 1) + 1];
Random random = new Random();
for (int i = 0, count = 0, t = random.nextInt(1000); i < arr.length - 1; i++, count++){
  //每k个步长更新一次数字
  if (count >= k){
      t = random.nextInt(1000);
      count = 0;
  }
  arr[i] = t;
}

//随机得到唯一不重复的数字
arr[arr.length - 1] = random.nextInt(1000);

System.out.println(Arrays.toString(arr));

//k个k进制的数各位不进位相加结果为0
int sum = 0;
Stack<Integer> stack = new Stack<Integer>();
StringBuilder sb = new StringBuilder();
for (int i = 0; i < arr.length; i++){
  //转为k进制
  int t = Integer.parseInt(Integer.toString(arr[i], k));

  //各位相加不进位，结果存入栈中
  while (t != 0){
      int x = (sum % 10) + (t % 10);
      x %= k;
      stack.push(x);
      t /= 10;
      sum /= 10;
  }
  //将结果转为字符串
  while (!stack.isEmpty()){
      sb.append(stack.pop());
  }
  //结果字符串转为数字
  sum = Integer.parseInt(sb.toString());
  //清空字符串备下次使用
  sb.delete(0, sb.length());
}
//将k进制结果转为10进制
int result = 0;
int p = 1;
while (sum != 0){
  result += (sum % 10) * p;
  p *= k;
  sum /= 10;
}
System.out.println(result);

      //利用HashMap求解
HashMap<Integer, Integer> map = new HashMap<Integer, Integer>();
for (int i = 0; i < arr.length; i++){
  //将相同key值即相等的数字出现个数存入对应value中
  if (map.get(arr[i]) != null){
      map.put(arr[i], map.get(arr[i]) + 1);
  }
  else{
      map.put(arr[i], 1);
  }
}
//遍历得到出现一次的键值，即为数组中出现一次的数字
Set<Integer> set = map.keySet();
for (Integer i : set){
  if (map.get(i) == 1){
      System.out.println(i);
      break;
  }
}
