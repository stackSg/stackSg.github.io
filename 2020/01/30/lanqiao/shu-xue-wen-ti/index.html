天平称重：变种三进制

  • 用天平称重时,我们希望用尽可能少的砝码组合称出尽可能多的重量。如果有无限个砝码,但它们的重量分别是1, 3, 9, 27, 81, ….等3的指数幂神奇之处在于用它们的组合可以称出任意整数重量(砝码允许放在左右两个盘中)。
    本题目要求编程实现:对用户给定的重量,给出砝码组合方案,重量< 1000000。
    例如:
    用户输入:
    5
    程序输出:
    9-3-1

  • 暴力法：

    public static void main(String[] args) {
  System.out.println(Integer.toString(1000000, 3));
  Scanner scanner = new Scanner(System.in);
  int N = scanner.nextInt();
  // m1(N);
  //转成3进制
  final String x = Integer.toString(N, 3);
  //翻转后转成字符数组
  char[] arr = new StringBuilder(x).reverse().toString().toCharArray();
  //容器放处理之后的0 -1 1
  List<Integer> list = new ArrayList<>();
  for (int i = 0; i < arr.length; i++) {
    if (arr[i] == '2') {
      list.add(0, -1);//-1插在开头
      if (i == arr.length - 1) {
        list.add(0, 1);//最后一个字符,进位
      } else {
        ++arr[i + 1];//否则,对下一个数字加1
      }
    } else if (arr[i] == '3') {
      list.add(0, 0);//插入0
      //更高位进1
      if (i == arr.length - 1) {
        list.add(0, 1);
      } else {
        ++arr[i + 1];
      }
    } else {
      list.add(0, arr[i] - '0');
    }
  }
  StringBuilder sb = new StringBuilder();
  for (int i = 0; i < list.size(); i++) {
    if (list.get(i) == 1) sb.append("+").append((int) Math.pow(3, list.size() - i - 1));
    if (list.get(i) == -1) sb.append("-").append((int) Math.pow(3, list.size() - i - 1));
  }
  System.out.println(sb.substring(1));
}

Nim游戏

  • 一共有N堆石子, 编号1..n ,第i堆中有个a[i]个石子。每一次操作Alice和Bob可以从任意-堆石子中取出任意数量的石子至少取一颗,至多取出这一堆剩下的所有石子。两个人轮流行动,取光所有石子的一-方获胜。Alice为先手。给定a,假设两人都采用最优策略,谁会获胜?

  • 解题算法：将当前剩余石子个数异或，面对非0局面为必胜。

  • 解：

    private static boolean win(int[] arr) {
  int res = 0;
  for (int i = 0; i < arr.length; i++) {
      res ^= arr[i];
  }

  return res != 0;
}

    public static void main(String[] args) {
  Scanner input = new Scanner(System.in);
  //接受测试用例个数
  int n = input.nextInt();
  String res = "";
  for (int i = 1; i <= n ;i++){
      //得到棋子个数
      int len = input.nextInt();
      //得到棋子位置
      int[] arr = new int[len];
      for (int j = 0; j < arr.length; j++) {
          arr[j] = input.nextInt();
      }
      //计算结果
      res += win(arr) + "\n";
  }
  //输出结果
  System.out.println(res);
}
private static String win(int[] arr) {
  //对输入数据排序
  Arrays.sort(arr);
  int res = 0;
  if (arr.length % 2 == 0){
      //长度为偶数
      for (int i = 1; i < arr.length; i+=2) {
          //得到两个棋子之间空缺的位置个数(相当于Nim游戏中的石子剩余个数)
          res ^= arr[i] - arr[i-1] - 1;    
      }
  }else{
      //长度为奇数
      for (int i = 0; i < arr.length; i+=2) {
          if (i == 0){
              //如果长度为奇数则第一个元素的"石子个数"为距离边界的个数
              res ^= arr[i] - 1;
          }else{
              res ^= arr[i] - arr[i-1] - 1;
          }
      }
  }

  if (res == 0){
      return "Bob will win";
  }else{
      return "Georgia will win";
  }
}

    //最大公约数  辗转相除
static int gcd(int m, int n) {
  if (n == 0) {
      return m;
  }

扩展欧几里得算法-裴蜀(贝祖)等式

  • 对任何整数a、b和它们的最大公约数d ,关于未知数x和y的线性丢番图方程(称为裴蜀等式) :ax + by = m有整数解时当且仅当m是d的倍数。裴蜀等式有解时必然有无穷多个整数解,每组解x、y都称为裴蜀数,可用扩展欧几里得算法(Extended Euclidean algorithm)求得。

  • 方程12x+42y=6有解

  • 特别地，方程ax+by=1有整数解当且仅当整数a和b互素

  • 推导过程：
    gcd(a,b)
    return b==0?a
    我们观察到：欧几里德算法停止的状态是： a’= gcd ， b’ = 0 ，(a’,b’是递归最后一层时参数的值)那么，这是否能给我们求解 x y 提供一种思路呢？
    a’x + b’y = gcd 此时x=1,y为任意数
    因为，这时候，只要 a = gcd 的系数是 1 ，那么只要 b 的系数是 0 或者其他值
    （无所谓是多少，反正任何数乘以 0 都等于 0 但是a 的系数一定要是 1），这时，我们就会有： a’1 + b’0 = gcd

    当然这是最终状态，但是我们是否可以从最终状态反推到最初的状态呢？

    假设当前我们要处理的是求出 a 和 b的最大公约数，并求出 x 和 y 使得 ax + by= gcd ……(1)，—>要求的
    而我们已经求出了下一个状态：b 和 a%b 的最大公约数，并且求出了一组x1 和y1 使得： bx1 + (a%b)y1 = gcd (2) ，–>下一个状态
    那么这两个相邻的状态之间是否存在一种关系呢？

    a%b = k ==> a = b(a/b) +k “/“舍掉余数的除法 ==> k=a-(a/b)b
    我们知道： a%b = a - (a/b)*b（这里的 “/” 指的是整除，例如 5/2=2 , 1/3=0），那么，我们可以进一步得到：

    gcd = bx1 + (a-(a/b)b)*y1

    = bx1 + ay1 – (a/b)by1

    = ay1 + b(x1 – a/b*y1) ……(3)

    对比之前我们的状态,式(3)和式(1)：求一组 x 和 y 使得：ax + by = gcd ，是否发现了什么？

    这里：

    x = y1

    y = x1 – a/b*y1

    这就是递推式,注意x,y是递归过程中的上一层,x1,y1是下一层(下一个状态)得到的值

  • 扩展欧几里得算法就是在求a,b的最大公约数m=gcd(a,b)的同时,求出贝祖等式ax + by= m的一个解(x0,y0)

  • 如何递推?
    X=y1
    y=x1- a/b*y1

  • 通解:
    x = x0+ (b/gcd)*t 所有的x对b同模
    y = y0- (a/gcd) *t 所有的y对a同模

  • 如果想要得到x大于0的第一个解?
    b/=d;x = (x0%b+b)%b

static long x;
static long y;
static long ext_Gcd(long a, long b){
    if (b == 0){
        x = 1;
        y = 0;
        return a;
    }
    long gcd = ext_Gcd(b, a % b);
    long t = x;
    x = y;
    y = t - a / b * y;

    return gcd;
}

static long linearIndefiniteEquation(long a, long b, long m) throws Exception{
    long gcd = ext_Gcd(a, b);
    if (m % gcd != 0){
        throw new Exception("无解");
    }

    long n = m / gcd;
    x = x * n;
    y = y * n;
    return gcd;
}

同余方程

模运算

  • 计算除以m的余数,叫做对m取模将a,b对m取模的结果相同,记为a=b(mod m),即a mod m==b mod m
  • 如果a=b(mod m),且c=d(mod m):
    a+b=c+d(mod m)
    ab=cd(mod m)

求解同余方程

  • linear congruence theorem
  • 用扩展欧几里德算法求解模线性方程的方法:
    同余方程ax=b (mod n)对于未知数x有解,当且仅当b是gcd(a,n)的倍数。且方程有解时,方程有gcd(a,n)个解。
  • 求解同余方程ax = b(mod n)相当于求解线性方程 ax + ny = b(x, y为整数)
    ax = ny1 + 余数
    b = ny2 + 余数
    ax - b = ny1 - ny2
    ax - b = n(y1 - y2)
    ax - b = ny(新的y中包含y1,y2)
    ax + ny = b(新的y中包含负号)

青蛙的约会

  • poj-1061
    设k为跳跃的次数
    mk + x = nk + y(mod L)
    (mk + x) + Ly2 = nk + y
    mk - nk + L    y2 = y - x
    (m - n)k + L*y2 = y - x
    得到线性不定方程，其中a = m - n, x = k, b = L, y = y2, m = y - x
public static void main(String[] args) {
    Scanner input = new Scanner(System.in);
    long x = input.nextLong();
    long y = input.nextLong();
    long m = input.nextLong();
    long n = input.nextLong();
    long L = input.nextLong();

    long a = m - n;
    long b = L;
    m = y - x;

    try {
        long gcd = 欧几里得扩展.linearIndefiniteEquation(a, b, m);
        //求得x,但是这里求的是第一个整数解，所以需要做转换
        x = 欧几里得扩展.x;
        //x增大或减小的单位是b/gcd为单位变化
        b /= gcd;
        b = Math.abs(b);
        //求得x的第一个正数解
        x = (x % b + b) % b;
        System.out.println(x);
    } catch (Exception e) {
        System.out.println("Impossible");
    }
}

素性测试

public class 素数 {
    //素性测试 O(n(1/2))
    static boolean is_prime(int n){
        for (int i = 2; i * i <= n; i++){
            if (n % i == 0)    return false;
        }
        return n != 1;
    }

    //约数枚举
    static List<Integer> divisor(int n){
        List<Integer> res = new ArrayList<Integer>();
        for(int i = 1; i * i <= n; i++){
            if (n % i == 0){
                res.add(i);
                if (i != n / i){
                    res.add(n / i);
                }
            }
        }
        return res;
    }

    //整数分解
    static Map<Integer, Integer> prime_factor(int n){
        Map<Integer, Integer> res = new HashMap<Integer, Integer>();
        for (int i = 2; i * i <= n; i++) {
            while (n % i == 0){
                Integer x = res.get(i);
                if (x != null){
                    res.put(i, x+1);
                }else{
                    res.put(i, 1);
                }
                n /= i;
            }
        }
        if (n != 1)    res.put(n, 1);
        return res;
    }
}

埃氏筛法

  • 前n个素数
static long prime(long n){
    //判断第n个素数的范围 
    //n范围内每个数的素数概率为1/ln(n) n / ln(n)是n范围内素数的个数
    int i = 2;
    while (i / Math.log(i) < n){
        i++;
    }
    //标记非素数
    int[] help = new int[i];
    for (int j = 2; j < help.length; j++) {
        if (help[j] == -1){
            continue;
        }

        int k = 2;
        while (k * j < i){
            help[k * j] = -1;
            k++;
        }
    }
    int count = 0;
    long res = 0;
    //统计所有的素数
    for (int j = 2; j < help.length; j++) {
        if (help[j] == 0){
            count++;
        }
        //找到第n个素数
        if (count == n){
            res = j;
            break;
        }
    }
    return res;
}

static int sieve(int n){
    prime = new int[n];
    isPrime = new boolean[n+1];
    for (int i = 0; i < isPrime.length; i++) {
        isPrime[i] = true;
    }
    isPrime[0] = isPrime[1] = false;
    int p = 0;
    for (int i = 2; i <= n; i++) {
        if (isPrime[i]){
            prime[p++] = i;
            for (int j = 2 * i; j <= n; j+=i) {
                isPrime[j] = false;
            }
        }
    }

    return p;
}

区间筛法

static boolean[] isPrime;
static boolean[] isPrimeSmall;
static long segmentSieve(long a, long b){
    isPrime= new boolean[(int) Math.sqrt(b)+1];
    isPrimeSmall = new boolean[(int) (b - a)];
    //[2,根号b)
    for (int i = 0; i < isPrime.length; i++) {
        isPrime[i] = true;
    }
    //[a, b)
    for (int i = 0; i < isPrimeSmall.length; i++) {
        isPrimeSmall[i] = true;
    }

    for (int i = 2; i * i <= b; i++) {
        if (isPrime[i]){
            //筛选[2,根号b)
            for (int j = 2 * i; j * j <= b; j+=i) {
                isPrime[j] = false;
            }
            //筛选[a, b)
            for (int j = (int) (Math.max(2l,(a+i-1)/i)*i); j < b; j+=i) {
                isPrimeSmall[(int) (j - a)] = false;
            }
        }
    }
    //统计个数
    long res = 0;
    for (int i = 0; i < isPrimeSmall.length; i++) {
        if (isPrimeSmall[i])
        res++;
    }
    return res;
}

快速幂运算

//将幂转为二进制计算
public class 快速幂运算 {
    //将幂转为二进制计算
    static long pow(long a, long b){
        long res = 1;
        long p = a;
        while (b != 0){
            if ((b & 1) == 1){
                res *= p;
            }
            p *= p;
            b >>= 1;
        }

        return res;
    }

    //矩阵幂运算求斐波那契
    static int fib(int n){
        if (n == 1 || n == 2)    return 1;
        int[][] matrix = {
                {0, 1},
                {1, 1}
        };

        int[][] res = matrixPow(matrix, n - 1);
        res = 矩阵乘法.matrixMultiply(new int[][]{{1, 1}}, res);
        return res[0][0];
    }
    //矩阵幂运算
    private static int[][] matrixPow(int[][] a, int b) {
        //得到单位矩阵
        int[][] res = new int[a.length][a[0].length];
        for (int i = 0; i < res.length; i++) {
            for (int j = 0; j < res[0].length; j++) {
                if (i == j){
                    res[i][j] = 1;
                }
            }
        }
        //平方
        int[][] p = a;

        while (b != 0){
            if ((b & 1) == 1){
                res = 矩阵乘法.matrixMultiply(res, p);
            }
            p = 矩阵乘法.matrixMultiply(p, p);
            b >>= 1;
        }

        return res;
    }

    public static void main(String[] args) {
        System.out.println(pow(2, 7));
        System.out.println(fib(15));
    }
}
