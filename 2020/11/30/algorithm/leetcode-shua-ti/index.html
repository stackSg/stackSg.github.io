  • 一个栈依次压入1、2、3、4、5，那么从栈顶到栈底分别为5、4、3、2、1。将这个栈转置后，从栈顶到栈底为1、2、3、4、5，也就是实现栈中元素的逆序，但是只能用递归函数来实现，不能用其他数据结构。

      public  static void reverse(Stack<Integer> stack){
      if (stack.isEmpty())    return;
      int n = getAndRemoveLastElement(stack);
      reverse(stack);
      stack.push(n);
  }

  public static int getAndRemoveLastElement(Stack<Integer> stack){
      Integer i = stack.pop();
      if (stack.isEmpty()){
          return i;
      }else {
          int n = getAndRemoveLastElement(stack);
          stack.push(i);
          return n;
      }
  }

  • 二叉树的遍历-Morris遍历：（核心是模拟了递归遍历三次回到当前节点的过程，但是morris只做到了两次）

    • 如果当前节点没有左孩子，向右移动
    • 如果当前节点有左孩子，左子树上最右的节点是否已经指向了当前节点：
      • 如果没指向，指向当前节点，向左移动
      • 如果指向，置为空，向右移动
    • 后续遍历：把打印时机放到第二次回到这个节点，然后逆序打印这个这个节点左子树的右边界。逆序打印通过反转链表实现。
