递归思想

  • 递归具有两层含义：
    1. 编程层面的递归调用
    2. 递推
  • 递归调用实际解决问题也是自下而上求解，只不过表现形式是自上而下
  • 递归是老板思维，迭代是员工思维
  • 递归调用的表现形式：
    1. 1.2.3路递归(通常是使用递推方式求出递推公式)
    2. 多路递归(循环)
    3. 封闭形式的直接解(根据公式直接求解)(未解决)

自上而下

  • 分治法：
    1. 分解
    2. 递归求解
    3. 合并
      快速排序、归并排序都属于分治法

自下而上

  • 递推：自下而上，由简到繁
  • 数学归纳：由简单推广到复杂
  • 迭代本质上也是递归，当然不是编程层面的递归调用，而是自下而上的递推过程。

    • 设想有个机器人坐在X*Y网格的左上角，只能向右、向下移动。机器人从(0,0)到(X,Y)有多少种走法?

    • 递归：

      static int fun(int x, int y) {
  if (x == 1 || y == 1) {
      return 1;
  }

  return fun(x, y - 1) + fun(x - 1, y);
}

    • 迭代：

      static int fun2(int x, int y) {
  int[][] squ = new int[x + 1][y + 1];

  // 写入初始数据
  for (int i = 1; i <= y; i++) {
      squ[1][i] = 1;
  }
  for (int i = 1; i <= x; i++) {
      squ[i][1] = 1;
  }

  // squ[i][j]=squ[i-1][j]+squ[i][j-1]
  for (int i = 2; i <= x; i++) {
      for (int j = 2; j <= y; j++) {
          squ[i][j] = squ[i - 1][j] + squ[i][j - 1];
      }
  }

  return squ[x][y];
}

  3. 硬币表示

  • 假设我们有8种不同面值的硬币｛1，2，5，10，20，50，100，200｝，用这些硬币组合构成一个给定的数值n。

    • 例如n=200，那么一种可能的组合方式为 200 = 3 * 1 + 1＊2 + 1＊5 + 2＊20 + 1 * 50 + 1 * 100.
    • 问总共有多少种可能的组合方式？ (这道题目来自著名编程网站ProjectEuler) 类似的题目还有：

    　　[华为面试题] 1分2分5分的硬币三种，组合成1角，共有多少种组合
    1x + 2y + 5*z=10
    　　[创新工厂笔试题] 有1分，2分，5分，10分四种硬币，每种硬币数量无限，给定n分钱，有多少组合可以组成n分钱

    1 5 10 25 分 n,多少种组合方法.

      //枚举
  public static int fun1(int[] arr, int n){
      int count = 0;
      //枚举每一个面值的0-最大值
      for (int i = 0; i <= n / arr[0]; i++) {
          for (int j = 0; j <= n / arr[1]; j++) {
              for (int k = 0; k <= n / arr[2]; k++) {
                  for (int m = 0; m <= n / arr[3]; m++) {
                      //如果所有硬币的总面值加起来正好等于n 就累加一种结果
                      if (i * arr[0] + j * arr[1] + k * arr[2] + m * arr[3] == n){
                          count++;
                      }
                  }
              }
          }
      }
      return count;
  }
  //递归
  static int fun2(int[] arr, int n, int cur){
      //如果只剩一种面值 则只有一种可能  直接返回
      if (cur == 0){
          return 1;
      }
      int res = 0;
      //枚举当前面值硬币的各种可能数，减去总钱数，减去当前面值，递归调用
      for (int i = 0; i * arr[cur] <= n; i++) {
          //累加结果
          res += fun2(arr, n - i * arr[cur], cur - 1);
      }
      return res;
  }
  //迭代
  static int fun3(int[] arr, int n){
      int[][] dp = new int[arr.length][n + 1];
      //初始化 在只有一种面值硬币的情况下，只有一种可能结果
      for (int i = 1; i < dp[0].length; i++) {
          dp[0][i] = 1;
      }
      //总钱数为0时  填充为1 方便后面取值
      for (int i = 0; i < dp.length; i++) {
          dp[i][0] = 1;
      }

      //动规
      int i = 0;
      int j = 0;
      for (i = 1; i < dp.length; i++) {
          for (j = 1; j < dp[0].length; j++) {
              int x = j / arr[i];
              //当前面值所有可能的个数
              for (int k = 0; k <= x; k++) {
                  //当前组合数+=总钱数-当前面值*当前个数
                  dp[i][j] += dp[i - 1][j - k * arr[i]];
              }
          }
      }

      return dp[dp.length - 1][dp[0].length - 1];
  }

  2. 非空子集-cc150 9.4

    • 编写一个方法，返回某集合的所用子集。

    • 递归：

      static Set<Set<Integer>> getSubSet(int[] a, int cur){
  Set<Set<Integer>> newSet = new HashSet<>();
  //如果只有一个元素则返回该元素组成的集合以及空集
  if (cur == 0){
      Set<Integer> nullSet = new HashSet<>();
      Set<Integer> set = new HashSet<>();
      set.add(a[0]);
      newSet.add(nullSet);
      newSet.add(set);
      return newSet;
  }

  //当前集合中的元素=上一集合中未添加当前元素的集合U上一集合中添加当前元素的集合
  Set<Set<Integer>> old = getSubSet(a, cur - 1);
  for (Set<Integer> set : old) {
      Set<Integer> clone = (Set<Integer>) ((HashSet<Integer>)set).clone();
      newSet.add(clone);
      set.add(a[cur]);
      newSet.add(set);
  }

  return newSet;
}

    • 迭代：

      static Set<Set<Integer>> getSubSet2(int[] a){
  Set<Set<Integer>> set = new HashSet<>();
  //初始化集合为空集
  Set<Integer> nullSet = new HashSet<>();
  set.add(nullSet);

  for (int i = 0; i < a.length; i++){
      //新集合=上一集合中未添加当前元素的集合U上一集合中添加当前元素的集合
      Set<Set<Integer>> newSet = new HashSet<>();
      for (Set<Integer> s : set) {
          Set<Integer> sClone = (Set<Integer>) ((HashSet)s).clone();
          Set<Integer> ss = (Set<Integer>) ((HashSet)s).clone();
          newSet.add(sClone);
          ss.add(a[i]);
          newSet.add(ss);
      }

      set = newSet;
  }

  return set;
}

    • 二进制法：

      //二进制法 当前元素是否加入集合正好可以用二进制1 0 分别表示
static ArrayList<ArrayList<Integer>> getSubSet3(int[] a){
  //保证入参有序
  Arrays.sort(a);

  ArrayList<ArrayList<Integer>> list = new ArrayList<>();
  //算得总子集合个数
  int n = (int) (Math.pow(2, a.length));
  //[集合总数,0)之间的数值二进制状态正好能够表示子集合的各种可能组合
  //由大到小取得的子集加入集合的顺序正好为字典序
  for (int i = n-1; i > 0; i--){
      //得到当前数值(状态)下的子集合
      ArrayList<Integer> newList = new ArrayList<>();
      //使用位运算取得每一位的状态
      for (int j = a.length-1; j >= 0; j--){
          if ((i >> j & 1) == 1){
              newList.add(a[j]);
          }
      }
      list.add(newList);
  }

  return list;
}

  3. 字符串(集合)全排列-cc150 9.5

    • 编写一个方法，确定某字符串的所有排列组合。

    • 插入法(迭代)：

      public static Set<String> fun(String s) {
  Set<String> set = new HashSet<>();
  //把第一个字符初始化到集合中
  set.add("" + s.charAt(0));
  //有多少个字符就需要多少次循环插入
  for (int i = 1; i < s.length(); i++) {
      Set<String> newSet = new HashSet<>();
      //取出当前集合中的所有元素，加入新的字符
      for (String str : set) {
          // 插到前面
          newSet.add(s.charAt(i) + str);
          //插到后面
          newSet.add(str + s.charAt(i));
          //插到中间
          for (int j = 1; j < str.length(); j++){
              newSet.add(str.substring(0, j) + s.charAt(i) + str.substring(j));
          }
      }
      set = newSet;
  }

  return set;
}

    • 插入法(递归)：

      //插入法-递归
public static Set<String> fun3(String s, int n){
  Set<String> newSet = new HashSet<>();
  //最简形式为只有一个字符，只需要将其加入集合返回
  if (n == 0){
      newSet.add("" + s.charAt(0));
      return newSet;
  }
  //得到之前的集合
  Set<String> old = fun3(s, n - 1);
  //将旧集合中的所有元素加上本位字符放入新集合
  for (String str : old) {
      // 插到前面
      newSet.add(s.charAt(n) + str);
      //插到后面
      newSet.add(str + s.charAt(n));
      //插到中间
      for (int j = 1; j < str.length(); j++){
          newSet.add(str.substring(0, j) + s.charAt(n) + str.substring(j));
      }
  }

  return newSet;
}

    • 交换法(多路递归)：

      //使用全局变量存放最终集合
static ArrayList<String> result = new ArrayList<>();
//交换法-递归
static void fun2(char[] arr, int k){
  //如果k=数组长度则证明当前组合下所有字符已经交换过
  if (k == arr.length){
      result.add(new String(arr));
  }

  //当前字符和数组每一个字符交换
  for (int i = k; i < arr.length; i++){
      //交换
      swap(arr, k, i);
      //交换剩余字符
      fun2(arr, k + 1);
      //还原/回溯
      swap(arr, k, i);
  }
}

//交换数组中两个元素
private static void swap(char[] arr, int k, int i) {
  char c = arr[k];
  arr[k] = arr[i];
  arr[i] = c;
}

    • 找出字典序的第k个排列

    • 前缀法(递归)：

      static int count = 0;
static int k = 3;
//前缀法求全排列字典序第k个元素
static void fun4(String prefix, char[] arr){
  //如果前缀长度等于原始数据长度则证明组合完成
  if (prefix.length() == arr.length){
      //统计完成个数
      count++;
      //打印第k项排列
      if (count == k){
          System.out.println("str(k) = " + prefix);
      }
  }
  //每次都遍历整个集合判断哪个元素未被加入前缀中
  for (int i = 0; i < arr.length; i++){
      char ch = arr[i];
      //如果字符在前缀中出现次数小于在原始集合中出现次数则证明未加入前缀
      if (count(prefix, ch) < count(new String(arr), ch)){
          //加入该字符后继续调用加入其他字符
          fun4(prefix + ch, arr);
      }
  }
}

//字符串中某字符出现的次数
private static int count(String str, char ch) {
  int count = 0;
  for (int i = 0; i < str.length(); i++){
      if (str.charAt(i) == ch){
          count++;
      }
  }
  return count;
}

深度优先搜索DFS

回溯

  • 递归调用代表开启一个分支，如果希望这个分支返回后某些数据恢复到分支开启前的状态以便重新开始,就要使用回溯技巧
  • 全排列的交换法,数独,部分和,用到了回溯

剪枝

  • 深搜时,如已明确从当前状态无论如何转移都不会存在(更优)解,就应该中断往下的继续搜索,这种方法称为剪枝
  • 数独里面有剪枝
  • 部分和里面有剪枝

      public static void main(String[] args) {
  //获取输入
  Scanner input = new Scanner(System.in);
  char[][] chess = new char[9][9];
  for (int i = 0; i < 9; i++) {
      String s = input.next();
      chess[i] = s.toCharArray();
  }
  //得到结果
  dfs(chess, 0, 0);
}

static void dfs(char[][] table, int x, int y) {
  //如果到第九行证明得到了结果
  if (x == 9) {
      //打印结果
      print(table);
      //因为答案唯一所以找到答案后直接退出
      System.exit(0);
  }
  //判断方格是否填入数字
  if (table[x][y] == '0') {
      //若未填入数字则从可能数字中依次寻找满足条件的数字
      for (int i = 1; i < 10; i++) {
          if (check(table, x, y, i)) {
              //找到后放入方格
              table[x][y] = (char) ('0' + i);
              //处理下一个状态
              dfs(table, x + (y + 1) / 9, (y + 1) % 9);
          }
      }
      //回溯，深度遍历完一条支路后需要进入下一个平行支路，此时需要将状态复原
      table[x][y] = '0';
  } else {
      //直接处理下一个状态
      dfs(table, x + (y + 1) / 9, (y + 1) % 9);
  }
}

//检查该方格能否填入数字i
private static boolean check(char[][] table, int x, int y, int i) {
  // 检查行
  for (int j = 0; j < 9; j++) {
      if (table[x][j] == '0' + i) {
          return false;
      }
  }
  // 检查列
  for (int j = 0; j < 9; j++) {
      if (table[j][y] == '0' + i) {
          return false;
      }
  }
  // 检查九宫格
  for (int j = x / 3 * 3; j < (x / 3 + 1) * 3; j++) {
      for (int k = y / 3 * 3; k < (y / 3 + 1) * 3; k++) {
          if (table[j][k] == '0' + i) {
              return false;
          }
      }
  }
  return true;
}

//打印
private static void print(char[][] table) {
  for (char[] cs : table) {
      for (char c : cs) {
          System.out.print(c);
      }
      System.out.println();
  }
}

  2. 部分和

    • 给定整数序列a1,a2,…,an,判断是否可以从中选出若干数,使它们的和恰好为k.
      1≤n≤20
      -10^8≤ai≤10^8
      -10^8≤k≤10^8
      样例:
      输入
      n=4
      a={1,2,4,7}
      k=13
      输出:
      Yes (13 = 2 + 4 + 7)

    • dfs方法：

      public static void main(String[] args) {
  //得到输入
  Scanner input = new Scanner(System.in);
  int n = input.nextInt();
  int[] arr = new int[n];
  for (int i = 0; i < arr.length; i++) {
      arr[i] = input.nextInt();
  }
  int k = input.nextInt();
  num = k;
  dfs(arr, k, 0, new ArrayList<>());
}
//拷贝数字用于打印
static int num = 0;
/**
* 
* @param arr 整数序列
* @param k 所要凑的数字
* @param cur 当前下标
* @param res 结果集
*/
static void dfs(int[] arr, int k, int cur, ArrayList<Integer> res){
  //如果出现恰好减到0的情况则打印并且退出
  if (k == 0){
      StringBuilder sb = new StringBuilder();
      sb.append("Yes(" + num + "=");
      for (Integer i : res) {
          sb.append(i + "+");
      }
      sb.deleteCharAt(sb.length() - 1);
      sb.append(")");
      System.out.println(sb);
      System.exit(0);
  }
  //如果k为负数或者当前下标越界则证明此分支下无正确结果需要返回
  if (k < 0 || cur == arr.length){
      return;
  }

  //不选当前元素
  dfs(arr, k, cur+1, res);
  //选择当前元素
  res.add(arr[cur]);
  int index = res.size() - 1;
  dfs(arr, k - arr[cur], cur+1, res);
  //回溯
  res.remove(index);
}

    • 二进制方法(迭代)

      //二进制方法
static void dfs2(int[] arr, int k){
  //算得集合可能的组合个数
  int n = (int) (Math.pow(2, arr.length) - 1);

  ArrayList<Integer> res = new ArrayList<>();
  //对每一种组合进行迭代
  for (int i = 0; i < n; i++){
      //截取每一二进制位状态
      for (int j = 0; j < arr.length; j++){
          //如果为1则加入结果集
          if ((i >> j & 1) == 1){
              res.add(arr[j]);
          }
      }
      //遍历结果集求得累加和
      int sum = 0;
      for (Integer ii : res) {
          sum += ii;
      }
      //如果累加和和所给整数相等则打印结果集并退出
      if (sum == k){
          StringBuilder sb = new StringBuilder();
          sb.append("Yes(" + num + "=");
          for (Integer ii : res) {
              sb.append(ii + "+");
          }
          sb.deleteCharAt(sb.length() - 1);
          sb.append(")");
          System.out.println(sb);
          return;
      }else{
          //如果不相等则排除此次组合，并清理结果集准备下一次组合
          res.clear();
      }
  }
}

      .W.W…..WW.
      W.W.W…..W.
      .W.W……W.
      ..W…….W.

      输出

      3

    • 解：

      public static void main(String[] args) {
  //得到输入数据
  Scanner input = new Scanner(System.in);
  int n = input.nextInt();
  int m = input.nextInt();
  char[][] arr = new char[n][m];

  for (int i = 0; i < arr.length; i++) {
      String s = input.next();
      arr[i] = s.toCharArray();
  }

  //如果找到一个点为水洼，则调用dfs将其八连通的水洼变为干地
  int count = 0;
  for (int i = 0; i < n; i++) {
      for (int j = 0; j < m; j++) {
          if (arr[i][j] == 'W') {
              dfs(arr, i, j);
              //统计调用dfs的次数即为水洼个数
              count++;
          }
      }
  }

  System.out.println(count);
}

public static void dfs(char[][] arr, int i, int j) {
  //首先将本水洼置为干地
  arr[i][j] = '.';

  //八连通的八个方向上只要有水洼则递归调用
  for (int x = -1; x < 2; x++) {
      for (int y = -1; y < 2; y++) {
          if (x == 0 && y == 0)
              continue;
          //判断下标是否越界
          if (i + x >= 0 && i + x < arr.length && j + y >= 0 && j + y < arr[0].length) {
              //判断下一位置是否为水洼
              if (arr[i + x][j + y] == 'W')
                  dfs(arr, i + x, j + y);
          }

      }
  }
}

  4. n皇后问题

    • 题目

    • 解：

      //存放每一行选择的列下标
static int[] rec;
//存放棋盘大小
static int n;
//存放总解数
static int count = 0;

static void dfs(int row){
  //如果行下标等于行数证明n个皇后已经放完
  if(row == n){
      //总数累加
      count++;
      //返回
      return;
  }
  //遍历本行每一列
  for (int col = 0; col < n; col++){
      //判断本行哪一格可以放皇后
      if (check(row, col)){
          //数组元素的下标代表第几行 值代表这一行放在了第几列
          rec[row] = col;
          //继续下一行
          dfs(row+1);
      }
  }
}
//判断皇后能否放入本单元格
private static boolean check(int row, int col) {
  //遍历当前行之前的行记录，判断是否和本单元冲突
  for (int i = 0; i < row; i++){
      //判断单元格所在列及所在正负对角线是否满足条件
      if (rec[i] == col || i - rec[i] == row - col || i + rec[i] == row + col){
          return false;
      }
  }

  return true;
}

  5. 素数环

    • 输入正整数n，对1-n进行排列，使得相邻两个数之和均为素数，输出时从整数1开始，逆时针排列。同一个环应恰好输出一次。
      n<=16
      如输入：6
      输出：
      1 4 3 2 5 6
      1 6 5 2 3 4

    • 解：

      //存放选中的数字
static int[] rec;
//存放输入数字
static int n;

static void dfs(int index) {
  //如果当前下标等于数组长度并且最后一个数和第一个数相加也为素数则找到一种解法
  if (index == rec.length && isP(rec[index - 1] + rec[0])) {
      //打印
      print();
      return;
  }
  //枚举每一个可选数字
  for (int i = 2; i <= n; i++) {
      //检查当前数字是否可选
      if (check(i, index)) {
          //将当前数字选入
          rec[index] = i;
          //继续挑选剩余数字
          dfs(index+1);
          //回溯
          rec[index] = 0;
      }
  }
}

//打印结果
private static void print() {
  StringBuilder s = new StringBuilder(n + "=");
  for (int i = 0;i < rec.length;i++){
      s.append(rec[i] + "+") ;
  }
  s.deleteCharAt(s.length() - 1);
  System.out.println(s.toString());
}

//检查当前数字是否可选
private static boolean check(int cur, int index) {
  //判断当前数字是否已经被选择
  for (int i = 0; i < index; i++) {
      if (rec[i] == cur) {
          return false;
      }
  }
  //判断当前数字和前一数字能否组成素数
  if (!isP(rec[index - 1] + cur)) {
      return false;
  }

  return true;
}

//判断数字是否为素数
private static boolean isP(int n) {

  for (int i = 2; i <= n / 2; i++) {
      if (n % i == 0)
          return false;
  }
  return true;
}

  6. 困难的串

    • 问题描述:如果一个字符串包含两个相邻的重复子串，则称它为容易的串，其他串称为困难的串,

    • 如:BB，ABCDACABCAB,ABCDABCD都是容易的，A,AB,ABA,D,DC,ABDAB,CBABCBA都是困难的。

      输入正整数n,L，输出由前L个字符(大写英文字母)组成的，字典序第n小的困难的串。
      例如，当L=3时，前7个困难的串分别为:
      A,AB,ABA,ABAC,ABACA,ABACAB,ABACABA
      n指定为4的话,输出ABAC

    • 解：

      public static void main(String[] args) {
  Scanner input = new Scanner(System.in);
  int n = input.nextInt();
  int L = input.nextInt();
  dfs(n, L, "");
}

//统计次数
static int count=0;
public static void dfs(int n, int L, String prefix) {
  //枚举字典序前L个字符进行选择
  for (char a = 'A'; a <= 'A' + L; a++){
      //判断前缀加入当前字符后是否为困难的串
      if (isHard(prefix, a)){
          //如果是困难的串则加入前缀字符串
          String s = prefix + a;
          //输出过程
          System.out.println(s);
          //累加次数
          count++;
          //如果到达指定次数则退出
          if (count == n){
              System.exit(0);
          }
          //继续加入字符
          dfs(n, L, s);
      }
  }
}
//判断加入字符a后字符是否为困难的串
private static boolean isHard(String prefix, char a) {
  int count = 1;
  //ABC[D] [i]-->A[BC][D i]-->...
  for (int i = prefix.length() - 1; i >= 0; i-=2){
      String s1 = prefix.substring(i, i + count);
      String s2 = "" + prefix.substring(prefix.length() - count + 1) + a;
      if (s1.equals(s2)){
          return false;
      }
      count++;
  }
  return true;
}
