3.6 奇安信 富途

基础题

  1. 哪种日志不会用于生产环境？
    OFF - FATAL - WARN - INFO - DEBUG - TRACE - ALL

  2. BGP协议中,哪种报文是用来传递路由的()。
    A.OPEN
    B.KEEPALIVE
    C.UPDATE
    D.NOTIFICATION

  3. 动态重定位需要重定位寄存器的支持

  4. 位图可用于哪些地方

  5. btree索引和hash索引优缺点

  6. 16进制转10进制

  7. static final
    static语法上可以重写，但没有重写效果，加上final后语法上禁止重写。

  8. 15层的台阶，每次最多跨3个，有多少种方法？

  9. Tcp第三次握手seq=1000,ack=2000.问第二次握手seq和ack分别为多少？

  10. 完全二叉树叶子节点的个数

  11. 给出中缀表达式，求后缀表达式

算法题

奇安信

  1. 奶牛产奶，给出两个数组，第一个数组为奶牛吃草量，第二个数组为奶牛产奶量。
    如果吃草多的奶牛产的奶比吃草少的奶牛少，那么这两个奶牛会打架。但如果两个奶牛吃草量相同或者产奶量相同则不会打架。
    问：奶牛之间不能打架，最多能产多少奶？

  2. 最短路算法

富途

  1. 在不改变入栈顺序的前提下，对栈中元素排序。
    [2, 1, 5, 3, 4] –> [5, 4, 3, 1, 2]

  2. 找出二叉查找树中两个错误节点。

  3. 信封嵌套的最大层数。如果一个信封的长度和宽度都小于另一个信封，则这两个信封可以嵌套。
    [[3,4],[2,3],[4,5],[1,3],[2,2],[3,6],[1,2],[3,2],[2,4]] = 4
    [[1,4],[4,1]] = 1

3.7

保融科技

  1. List<Object>引用不能接收List<String>类型对象
  2. 哈夫曼编码 加权路径和
  3. SQL语句大题
  4. 手写设计模式 观察者 命令模式 责任链模式 迭代器模式选其一。
  5. 生产者消费者模式

字节跳动

  1. 找到数组中比当前值大的在当前值右边最近的数字，如果没有显示-1.
  2. n个人围成一圈，给出身高数组，求最小身高差下的排列的最大差值。
  3. 书架摆书。并查集
  4. 最小编辑距离类似，求能通过前一个字符串一次改变为当前字符串的最长子序列的长度。

3.11

安硕信息

  1. 约瑟夫环问题
  2. Properties是线程安全的
  3. ByteArrayOutputStream字节数组输出流在内存中创建一个字节数组缓冲区
  4. 线程通过yield()方法使具有相同优先级线程获得处理器

跟谁学

  1. 在跟谁学学习的途途小朋友有一天向老师求助，原来途途的父母要出差n天，走之前给途途留下了m个棒棒糖。
    途途决定每天吃的棒棒糖数量不少于前一天吃的一半，但是他又不想在父母回来之前的某一天没有棒棒糖吃，
    他想让老师帮他计算一下，他第一天最多能吃多少个棒棒糖。
  2. 不适用库函数计算一个整数的平方根。（二分法）
  3. 求矩阵的转置。

3.12

西安立辰远景

  1. [i, j, k] 给数组中i-j的区间+k，有很多个三元组进来，求最后的结果。
  2. a_time = [1, 2, 3, 4, 4, 8], p_time = [50, 20, 11, 25, 30, 40]
    食堂排队买饭，a_time是到食堂的时间，p_time是买饭花费时间，食堂有四个窗口，问所有人平均等待时间。
  3. 帖子禁封，违规默认封60秒，a数组是禁封时间点，b数组是管理员手动解封时间点，问改用户被禁封的总时长。
  4. 面试题 17.21. 直方图的水量

3.13

美团

  1. 就是一个n行m列矩阵，让你输出行列翻转。
    输入:
    3 3
    1 2 3
    4 5 6
    7 8 9
    输出：
    1 4 7
    2 5 8
    3 6 9

  2. 给你一个字符串，你提取数字后，对数字进行从小到大输出，数字要去除前导0

  3. 给n个数，求每个窗口大小为m的众数是多少 输出n-m+1行

  4. 给一棵树，每个点有权值，让选择点，要求选择不相邻的点，求使得权值最大，在使得权值最大的情况下，求选择的点中最小的权值最大是多少

  5. 给你一个图，每个点有自己的权值，现在就是说你可以选择任意一个点开始，然后你走的话，只能走到点权比当前小的点。
    问最多能走多少个点。

  3. 平均等待时间 [[1,2],[1,3],[4,3]]

  6. 常见的对称加密算法：DES、3DES、AES、DESX、Blowfish、、RC4、RC5、RC6；
    常见的非对称加密算法：RSA、DSA（数字签名用）、ECC（移动设备用）、Diffie-Hellman、El Gamal。
    常见的散列算法（Hash算法—单向加密算法）：MD5、SHA、
    对称与非对称加密算法
  7. 什么是SQL注入攻击，如何预防？

德科-华为研发

  1. 给出一个有重复字符的字符串，求重复字符数量第k大的子串。HHHHAABBBC 3 -> 2
  2. 数字编码，数字二进制表示中低7位+一位 这一位为1代表后续还有高位 为0表示后续没有高位，组成8位后将其转为16进制并大写，以小端方式放在左边。0011 11101000 -> E807
  3. 输入一组数字，输出其所能排列出的最大值。
