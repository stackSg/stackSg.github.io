相关概念

  • 刚开始计算机都是处理数值数据，后来引入字符串的概念，计算机可以处理非数值的数据了，当然原理是使用数值模拟非数值数据（通过字符编码表，例如ASCII）。
  • 字符串也称为串。
  • 子串和主串。”ABC”是”ABCDEF”的子串，反之则倒过来。
  • 字符串比较。比较对应位置ASSCII码值的大小。比如”abc”和”acc”中’b’<’c’则字符串”abc”<”acc”。
  • 字符串一般都是比较是否相等。
  • 子串定位操作称为串的模式匹配。

字符串匹配算法

BF算法

  • BF算法，即暴风(Brute Force)算法，是普通的模式匹配算法。BF算法的思想就是将目标串S的第一个字符与模式串T的第一个字符进行匹配，若相等，则继续比较S的第二个字符和 T的第二个字符；若不相等，则比较S的第二个字符和T的第一个字符，依次比较下去，直到得出最后的匹配结果。BF算法是一种蛮力算法。

  • 代码示例

      /**
      str:目标串
      t:模式串
      pos:指定目标串的起始下标 
      return:若匹配成功则返回相应开始位置（从1开始），失败返回0 
  **/
  int BF(char *str, char *t, int pos){
      int i, j;
      if (0 <= pos && pos <= str[0]){
          i = pos;
          j = 1;
      }else{
          return 0;
      }

      while (i <= str[0] && j <= t[0]){    //数组第一个位置存放字符串长度 
          if (str[i] == t[j]){
              i++;
              j++;
          }else{                //匹配失败则回退 
              i = i - j + 2;    
              j = 1;
          }
      }

      if (j > t[0]){
          return i - t[0];
      }else{
          return 0;
      }
  }

  • 该算法最坏情况下要进行M(N-M+1)次比较，时间复杂度为O(MN)。

Rabin Karp

  • 使用哈希函数算出子串的hash值，然后在目标串中寻找匹配的hash值即能找到匹配的子串。

  • hash，复杂度O(M*N):

      static long seed = 31;

  static long hash(String s){
      long hash = 0;

      for (int i = 0; i < s.length(); i++) {
          hash = hash * seed + s.charAt(i);
      }

      return hash;
  }

  static void RabinKarp(String p, String s){
      long hash = hash(p);

      for (int i = 0; i + p.length() <= s.length(); i++){
          long t = hash(s.substring(i, i + p.length()));
          if (t == hash){
              System.out.println("match_index:" + i);
          }
      }
  }

  • 优化(滚动hash,复杂度O(N)):

      /优化：滚动hash
  static void RabinKarp2(String p, String s){
      int n = p.length();
      long pHash = hash(p);
      long hash = hash(s.substring(0, n));

      if (hash == pHash){
          System.out.println("match_index:" + 0);
      }

      long t = hash;
      for (int i = n; i < s.length(); i++){
          //新的hash=加入新字符后的hash-头一个字符的hash
          t = ((t * seed + s.charAt(i)) - (s.charAt(i - n) * (long) Math.pow(seed, n)))%Long.MAX_VALUE;

          if (t == pHash){
              System.out.println("match_index:" + (i - n + 1));
          }
      }
  }

KMP算法

  • KMP算法是一种改进的字符串匹配算法，由D.E.Knuth，J.H.Morris和V.R.Pratt提出的，因此人们称它为克努特—莫里斯—普拉特操作（简称KMP算法）。KMP算法的核心是利用匹配失败后的信息，尽量减少模式串与主串的匹配次数以达到快速匹配的目的。具体实现就是通过一个next()函数实现，函数本身包含了模式串的局部匹配信息。

  • 代码示例

      /**
      t:模式串
      next:部分匹配表 
  **/
  //根据模式串建立部分匹配表 
  void GetNext(char t[], int next[]){
      int i = 1;    //i指后缀 
      int j = 0;    //j指前缀

      next[1] = 0;    //next[0]对应t[0]位置的字符串长度，不需要填值，next[1]前面没有值，默认为0 
      while (i <= t[0]){
          if (0 == j || t[i] == t[j]){
              i++;
              j++;
              next[i] = j;
          }else{
              j = next[j];
          }
      } 
  }

  /**
      str:目标串
      t:模式串
      next:部分匹配表
      pos:指定目标串起始匹配地址 
  **/ 
  int KMP(char *str, char *t, int *next, int pos){
      int i, j;
      if (0 <= pos && pos <= str[0]){
          i = pos;
          j = 1;
      }else{
          return 0;
      }

      while (i <= str[0] && j <= t[0]){    //str[0]和t[0]第一个位置存放字符串长度 
          if (0 == j || str[i] == t[j]){
              i++;
              j++;
          }else{                //匹配失败则回退 ,KMP是对BF的改进，在此处不需要回退到起始位置比较，而是通过 
              j = next[j];    //事先建立好的部分匹配表(next数组)将j回退到指定位置，提高效率 
          }
      }

      if (j > t[0]){
          return i - t[0];
      }else{
          return 0;
      }
  }

  • 后来有人发现，KMP算法是有缺陷的。比如主串”aaaaabcd”,子串”aaaaae”,得到的next数组为”012345”。往前回溯时会依次回退一个元素，但前面的a元素全都相等，则可以直接回退到第一个a元素。

  • 改进代码

      void GetNext(char t[], int next[]){
      int i = 1;    //i指后缀 
      int j = 0;    //j指前缀

      next[1] = 0;    //next[0]对应t[0]位置的字符串长度，不需要填值，next[1]前面没有值，默认为0 
      while (i <= t[0]){
          if (0 == j || t[i] == t[j]){
              i++;
              j++;
              if (t[i] == t[j]){            //判断如果前缀后缀相等，则next[i]直接指向前缀所指向的 
                  next[i] = next[j];
              }else{
                  next[i] = j;
              }
          }else{
              j = next[j];
          }
      } 
  }
